Profile Parser

The goal of the ProfileParser is to transform the XML record dump(s) from EasyDB platform into another XML documents compliant with the CIDOC-CRM entities structure. The resulting XML file can be used as input schema by X3M mapping tool to create the final RDF knowledge graph according to CIDOC-CRM ontologies.

Profile Mapper

The ProfileMapper tool is used to transform the XML records dump(s) from EasyDB platform into the target .3m file needed to map XML records into the CIDOC-CRM E22_Man-Made_Object knowledge graph.
